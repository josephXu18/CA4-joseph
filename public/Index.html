Designing this page

Edited on: October 20, 2024

I built this site using HTML, CSS, and JavaScript for the front end, with Node.js and Socket.io for the backend functionality. The structure is crafted with HTML, while CSS provides styling for a clean, professional look. I used JavaScript to add interactivity, such as a chat room feature, where I implemented real-time messaging with Socket.io to enable user connections, display active users, and manage typing notifications. Node.js serves as the server, handling requests and storing message history so new users can see past conversations.
